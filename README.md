# Features
- **Chain Tracking:** Hooks and inspects HTTP responses to track chain events.
- **Translations:** Uses Hachimi's translations (if available) to translate character names.

# How to Install

## Prebuilt
1. Download the plugin [here](https://github.com/xi-pec/chain-tracker/releases/latest)
2. Install the plugin using [this guide](https://hachimi.noccu.art/docs/plugins/installation).

## From source
1. Clone the repository using `git clone https://github.com/xi-pec/chain-tracker`
2. Build using `cargo build --release`
3. Locate DLL at `target/release/`
4. Install the plugin using [this guide](https://hachimi.noccu.art/docs/plugins/installation).
